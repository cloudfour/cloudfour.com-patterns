# Table

The `c-table` class applies default styling to HTML `table` elements.
A table can optionally have a header, a footer, and a caption.

<Canvas>
  <Story name="Basic">
    {basicDemo({
      hasHeader: boolean('Include Header', true),
      hasFooter: boolean('Include Footer', true),
      hasCaption: boolean('Include Caption', true),
      isRuled: boolean('Add c-tabled--ruled modifier to the table', false),
      isStriped: boolean('Add c-tabled--striped modifier to the table', false),
      numericData: boolean('Add c-table--numeric modifier to the table', false),
      tableData: data,
    })}
  </Story>
</Canvas>

## Numeric Data

Tables with lots of numeric data use a couple of modifications to make them
easier to read and understand.

Numeric cells are right-aligned and use tabular figures to make it easier to
compare cells across a column.

These changes can be achieved by adding c-table--numeric to the table.
