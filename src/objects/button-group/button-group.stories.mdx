# Button Group

The Button Group layout object can be used for aligning a group of related buttons.

## Default

<Canvas>
  <Story name="Default">{(args) => demo({ ...args })}</Story>
</Canvas>

## Tight spacing modifer

The `o-button-group--tight` modifer will tighten up the spacing between the buttons.

<Canvas>
  <Story name="Tight Spacing" args={{ class: 'o-button-group--tight' }}>
    {(args) => demo({ ...args })}
  </Story>
</Canvas>

## No stack modifier

By default buttons are stacked vertically at smaller viewports.
The `o-button-group--no-stack` modifier will remove this behavior.

<Canvas>
  <Story name="No Stack" args={{ class: 'o-button-group--no-stack' }}>
    {(args) => demo({ ...args })}
  </Story>
</Canvas>
