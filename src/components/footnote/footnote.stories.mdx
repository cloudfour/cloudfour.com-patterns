# Footnote

Used for displaying the content of a footnote.

Renders as a list item (`<li>`) Expected to be wrapped in a Footnote List (TODO: link)

    {(args) => template(args)}
  </Story>

## Template Properties

- `count` (number)

## Template Blocks

- `content` The footnote contents
- `id` (string) a unique ID for a footnote. Defaults to the `count` but can be customized if there are multiple sets of footnotes on one page
