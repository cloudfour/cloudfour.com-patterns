# Figure

The `c-figure` class applies styling to the figure element.

<Preview>
  <Story name="Basic">
    {basicDemo({
      src: 'https://picsum.photos/600/300',
      alt: 'This is an image'
    })}
  </Story>
</Preview>

## Figure with caption

Figures may be accompanied by an optional caption.

<Preview>
  <Story name="With Caption">
    {captionDemo({
      src: 'https://picsum.photos/600/300',
      alt: 'This is an image',
      hasCaption: boolean('Include Caption', true)
    })}
  </Story>
</Preview>
