# Table Object

The `o-table` class applies default styling to HTML `table` elements.
A table can optionally headers and footers.

<Preview>
  <Story name="Basic">
    {basicDemo({
      hasHeader: boolean('Include Header', true),
      hasFooter: boolean('Include Footer', true),
      rowModifier: select('Table Row Modifier Class', [
        '',
        'o-table--ruled',
        'o-table--striped'
      ]),
      numericCells: boolean(
        'Add o-table__cell--numeric modifier to cells',
        false
      ),
      rightAlignRowHeadings: boolean('Right-align row Headings:', false)
    })}
  </Story>
</Preview>

# Numeric Data

When cells contain numeric data, aligning them to the right and using tabular
figures can make it easier to compare cells across a column. This can be
achieved by adding o-table\_\_cell--numeric to table cells.

<Preview>
  <Story name="Numeric Data">
    {basicDemo({
      hasHeader: boolean('Include Header', true),
      hasFooter: boolean('Include Footer', true),
      rowModifier: select('Table Row Modifier Class', [
        '',
        'o-table--ruled',
        'o-table--striped'
      ]),
      numericCells: true,
      rightAlignRowHeadings: boolean('Right-align row Headings:', false)
    })}
  </Story>
</Preview>

# Tighten Row Headings

In complex tables, row headings sometimes begin to feel disconnected from the
data in their row. You can tighten up the visual connection by right-aligning
row headings. To do so, add the `o-table--tighten-row-headings` to the table.

<Preview>
  <Story name="Tighten Row Headings">
    {basicDemo({
      hasHeader: boolean('Include Header', true),
      hasFooter: boolean('Include Footer', true),
      rowModifier: select('Table Row Modifier Class', [
        '',
        'o-table--ruled',
        'o-table--striped'
      ]),
      numericCells: boolean(
        'Add o-table__cell--numeric modifier to cells',
        false
      ),
      rightAlignRowHeadings: true
    })}
  </Story>
</Preview>
