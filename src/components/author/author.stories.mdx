# Author

Displays an author, or set of authors, for article bylines.

## Single Author

### With `date` defined

Will display the `date` when it is defined.

<Canvas>
  <Story name="With date" args={{ count: 1 }}>
    {(args) =>
      template({
        authors: authors(args.count),
        date: new Date('March 31, 2021'),
      })
    }
  </Story>
</Canvas>

### Without `date` defined

Will display the author `job_title` when `date` is not defined.

<Canvas>
  <Story name="Without date" args={{ count: 1 }}>
    {(args) =>
      template({
        authors: authors(args.count),
      })
    }
  </Story>
</Canvas>

## Template Properties

`class`: Append a class to the root element.

`authors`: Array of [author objects](https://timber.github.io/docs/reference/timber-user/#properties) of the type:

```ts
 {
   name: string | () => string;
   avatar: string | () => string;
   link: string | () => string;
   job_title: string | () => string;
 }
```

`date` (optional): A date object representing the publication date
