# Button Group

The Button Group object pattern can be used for a collection of buttons.

## Default spacing

Add the `o-button-group` CSS class to the wrapper container.

```html
<div class="o-button-group">
  <ul role="list">
    <li>
      <button class="c-button">Button 1</button>
    </li>
    <li>
      <button class="c-button">Button 2</button>
    </li>
    <li>
      <button class="c-button">Button 3</button>
    </li>
  </ul>
</div>
```

## Tight spacing

There is also a modifier, `o-button-group--tight`, to tighten up the spacing between the buttons.

```html
<div class="o-button-group o-button-group--tight">
  <ul role="list">
    <li>
      <button class="c-button">Button 1</button>
    </li>
    <li>
      <button class="c-button">Button 2</button>
    </li>
    <li>
      <button class="c-button">Button 3</button>
    </li>
  </ul>
</div>
```
