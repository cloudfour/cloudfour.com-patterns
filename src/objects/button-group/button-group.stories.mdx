# Button Group

The Button Group layout object can be used for aligning a group of related buttons.

## Default

  <Story name="Default">{(args) => demo({ ...args })}</Story>

## Tight spacing modifer

The Button Group has a tight-spacing modifier.
You can apply it via the `tight` boolean argument:

```php
{% embed '@cloudfour/objects/button-group/button-group.twig' with { tight: true } %}
  ...
{% endembed %}
```

<Canvas>
  <Story name="Tight Spacing" args={{ count: 5, tight: true }}>

  </Story>
</Canvas>

## Template Properties

- `class`: Append a class to the root element.
- `content` (block): Block to add Button elements.
- `tight` (boolean, default: `false`): Adds the `o-button-group--tight` CSS class modifier.
