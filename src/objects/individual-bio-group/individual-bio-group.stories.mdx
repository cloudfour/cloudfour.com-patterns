# Individual Bio Group

The Individual Bio Group layout object can be used for the individual bio pages. It helps layout the bio avatar, content and optional bio links.

## Template Properties

- `avatar` (block): Block to add a bio avatar.
- `content` (block): Block to add the main bio content.
- `links` (block): Optional block to bio links.
