# Comment Form

The form used to add a comment to an article.

<Canvas>
  <Story name="Default">
    {({ isLoggedIn }) =>
      template(isLoggedIn ? { loggedInUser: tyler, logOutUrl: '/logout' } : {})
    }
  </Story>

## Template Properties

- `class`: Append a class to the root element.
- `loggedInUser`: [user object](https://timber.github.io/docs/reference/timber-user/#properties) of the type:

  ```ts
  {
    name: () => string;
    link: () => string;
  }
  ```

- `logOutUrl`: URL used for log out link.
