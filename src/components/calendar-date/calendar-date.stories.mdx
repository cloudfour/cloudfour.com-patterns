# Calendar Date

Formats a `<time>` element so that it looks like a calendar icon. Intended to accompany upcoming presentations and events.

A short note may be included, for example the event length ("3-day event"), type ("teleconference") or sales status ("sold out").

<Canvas>
  <Story
    name="Basic"
    argTypes={{
      datetime: {
        defaultValue: Number(new Date()),
        control: { type: 'date' },
      },
    }}
  >
    {({ datetime }) => template({ datetime: new Date(datetime) })}
  </Story>
  <Story
    name="With Note"
    argTypes={{
      datetime: {
        defaultValue: Number(new Date()),
        control: { type: 'date' },
      },
      note: {
        type: { name: 'string', required: false },
        defaultValue: '3-day event',
      },
    }}
  >
    {({ datetime, note }) =>
      template({
        datetime: new Date(datetime),
        note: note,
      })
    }
  </Story>
</Canvas>

## Seasons

To add some fun and visual variety, each season has its own header color. These are applied using a modifier class based on the month, for example `c-calendar-date--january`.
