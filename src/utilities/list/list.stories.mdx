# List

Utilities for tweaking the appearance and behavior of `<ul>` and `<ol>` elements.

## Style Type

- `u-list-none` removes bullets or numbering from a list. When using this utility on a `<ul>`, it's important to include `role="list"` to avoid [an issue with unstyled lists and Safari/VoiceOver](https://www.scottohara.me/blog/2019/01/12/lists-and-safari.html).
