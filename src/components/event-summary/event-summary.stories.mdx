# Event Summary

Combines the [Media object](/docs/objects-media--image) and [Calendar Date component](/docs/components-calendar-date--basic) to display information for an upcoming speaking event.

<Canvas>

      href: 'https://cloudfour.com',
    }}

    {(args) => template(args)}
  </Story>
</Canvas>

## Template Properties

- `class` (string): Append a class to the `c-event-summary` root element.
- `heading_level` (number, default `3`): The header heading level (e.g. `2` renders an `<h2>` element).
- `datetime` (string): Refer to the [Calendar Date component](/docs/components-calendar-date--basic#template-properties)
- `event` (string): The name of the event (e.g. `'Smashing Magazine'`)
- `href` (string): The event URL (e.g. `https://cloudfour.com`)
- `location` (string): The location of the event (e.g. `'Portland, OR'`)
- `note` (string): A short event note (e.g. "3 day event")
- `speakers` (string array): An array of string names of the speakers:
  ```ts
  ['Megan', 'Aileen'];
  ```

## Template blocks

- `note` (optional): Refer to the [Calendar Date component](/docs/components-calendar-date--basic#template-blocks)
- `event_details` (optional): Overrides default event details section (location & speakers)
