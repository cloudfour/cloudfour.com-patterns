# Icon Object

The `o-icon` class causes an SVG element to inherit its size and default color from its parent element's `font-size` and `color` value.

For a list of available icons, visit [the Icons page](/?path=/docs/icons--page).

## Modifier: Inline

Icons without a flex or grid container may appear [slightly low in comparison to adjacent text](https://cloudfour.com/thinks/some-imaginary-css/#align-to-typeface-median). The `o-icon--inline` modifier applies a slight positioning adjustment to account for this.
