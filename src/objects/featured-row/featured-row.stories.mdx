# Featured Row

The Featured Row layout object can be used by individual sections on a page to place content. It is helpful for highlighting a section by giving it a distinct layout. It includes a heading, main content area, and an optional link.

<Canvas>
  <Story name="Default">{defaultDemo}</Story>
</Canvas>

## Template Properties

- `heading` (block): Block to add the heading.
- `content` (block): Block to add the main content.
- `action` (block): Optional block to add a cta.
