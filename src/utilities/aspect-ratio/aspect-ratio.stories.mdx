# Aspect Ratio

These utilities apply aspect ratios. They may be used to visually offset content from its surroundings. For example, a screenshot in an article with a white background might blend in too much with the page without a border.

## Sizes

Available sizes are based on [our `size.$edge-` tokens](/?path=/docs/design-tokens-size--page#edge):

- `u-aspect-ratio-square`
- `u-aspect-ratio-full`
- `u-aspect-ratio-wide`
- `u-aspect-ratio-cinema`

## Responsive aspect ratios

All of these utilities include responsive variations. Appending `@{breakpoint}` to the end of the class name will cause the utility to change for the given breakpoint.

## Custom aspect ratios

The `--aspect-ratio` CSS custom property can be used to customize the aspect ratio.
