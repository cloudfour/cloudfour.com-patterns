# Speaking Event

Displays information for an upcoming speaking event. Combines the Media and Calendar Date components for this specific usecase.

<Canvas>
  <Story
    name="Multiple speakers"
    args={{
      datetime: '2021-10-03',
      event: 'Smashing Magazine',
      note: '3 day event',
      location: 'New York, NY',
      speakers: ['Megan', 'Aileen'],
    }}
  >
    {(args) => template(args)}
  </Story>
  <Story
    name="Single speaker"
    args={{
      datetime: '2021-12-09',
      event: 'An Event Apart',
      location: 'Seattle, WA',
      speakers: ['Arianna'],
    }}
  >
    {(args) => template(args)}
  </Story>
</Canvas>

## Template Properties

- `class` (string): Append a class to the `c-avatar` element.
- `datetime` (string): Refer to the [Calendar Date component](/?path=/docs/components-calendar-date--basic)
- `note` (block): Refer to the [Calendar Date component](/?path=/docs/components-calendar-date--basic)
- `event` (string): The name of the event (e.g. `'Smashing Magazine'`)
- `location` (string): The location of the event (e.g. `'Portland, OR'`)
- `speakers` (string array): ...
