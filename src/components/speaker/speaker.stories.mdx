# Speaker

Displays a speaker, or set of speakers, for ~~article bylines~~.

<Canvas>
  <Story name="Default">
    {(args) =>
      template({
        speakers: [megan, tyler, gerardo].slice(0, args.count),
      })
    }
  </Story>
</Canvas>

## Template Properties

- `class`: Append a class to the root element.
- `speakers`: Array of [author objects](https://timber.github.io/docs/reference/timber-user/#properties) of the type:

  ```ts
  {
    name: string | () => string;
    avatar: string | () => string;
    link: string | () => string;
  }
  ```

- `date`: A date object representing the publication date
