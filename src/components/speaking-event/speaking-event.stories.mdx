# Speaking Event

Combines the [Media object](/docs/objects-media--image) and [Calendar Date component](/docs/components-calendar-date--basic) to display information for an upcoming speaking event.

<Canvas>
  <Story
    name="Multiple speakers"
    args={{

      note: '3 day event',
      location: 'New York, NY',
      speakers: ['Megan', 'Aileen'],
      href: 'https://cloudfour.com',
    }}
  >
    {(args) => template(args)}
  </Story>
</Canvas>

## Template Properties

- `class` (string): Append a class to the `c-speaking-event` root element.
- `datetime` (string): Refer to the [Calendar Date component](/docs/components-calendar-date--basic#template-properties)
- `note` (block): Refer to the [Calendar Date component](/docs/components-calendar-date--basic#template-blocks)
- `event` (string): The name of the event (e.g. `'Smashing Magazine'`)
- `location` (string): The location of the event (e.g. `'Portland, OR'`)
- `speakers` (string array): An array of string names of the speakers:
  ```ts
  ['Megan', 'Aileen'];
  ```
