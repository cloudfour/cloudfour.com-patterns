# Button Group

The Button Group object pattern can be used for a collection of buttons.

## Default

<Canvas>
  <Story name="Default">{(args) => demo({ ...args })}</Story>
</Canvas>

## Tight spacing modifer

The `o-button-group--tight` modifer will tighten up the spacing between the buttons.

<Canvas>
  <Story name="Tight Spacing" args={{ class: 'o-button-group--tight' }}>
    {(args) => demo({ ...args })}
  </Story>
</Canvas>

## No stack modifier

The `o-button-group--no-stack` modifier will not stack the buttons vertically at smaller viewports (by default, buttons are stacked vertically at smaller viewports).

<Canvas>
  <Story name="No Stack" args={{ class: 'o-button-group--no-stack' }}>
    {(args) => demo({ ...args })}
  </Story>
</Canvas>
