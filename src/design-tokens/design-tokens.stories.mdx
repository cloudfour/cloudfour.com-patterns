## Extended Color Palette

The extended palette is intended to be ever evolving, it offers an additional range of colors work with. Each color has 5 values, with 1 being the lightest and 5 being the darkest. The number of different values was determined by cataloging the current use of color throughout our site to create a sufficient number of variations. Usage of these colors will vary, but they should come in handy for illustrations and components.
