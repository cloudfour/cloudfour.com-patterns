# Speaker

Displays a speaker, or a set of speakers.

## Single Speaker

A single speaker displays the avatar, linked speaker name and speaker title.

<Canvas>
  <Story name="Single Speaker" args={{ count: 1 }}>
    {(args) =>
      template({
        speakers: [arianna, danielle, megan].slice(0, args.count),
      })
    }
  </Story>
</Canvas>

## Multiple Speakers

The speakers' titles are not displayed when multipe speakers are rendered.

### Two Speakers

<Canvas>
  <Story name="Two Speakers" args={{ count: 2 }}>
    >
    {(args) =>
      template({

      })
    }
  </Story>
</Canvas>

## Template Properties

`class`: Append a class to the root element.

`speakers`: Array of [author objects](https://timber.github.io/docs/reference/timber-user/#properties) of the type:

```ts
{
  name: string | () => string;
  avatar: string | () => string;
  link: string | () => string;
  job_title: string | () => string;
}
```
