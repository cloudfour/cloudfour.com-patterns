# Media Link

The Media Link component extends [the Media object](/docs/objects-media--image) to present a small link with an accompanying image and (optionally) an iconographic affordance. It's useful for capping off a section of content with a link to a related resource: For example, a marketing blurb that links to a case study.

The visual object should reduce well while representing the content source: For example, an author image or app icon.

If you need more text or more complex visual objects, consider using a [Media Summary](/docs/components-media-summary--event) or [Card](/docs/components-card--content-blocks) instead.

## Template Blocks

In addition to the template properties listed above, you may specify the following blocks:

- `content`: The content of the link. If not specified, the `label` property will be used.
- `object`: The visual object. If not specified, the `avatar` property will be passed to [the Avatar component](/?path=/docs/components-avatar--empty) (along with other `avatar_*` properties).
