## Compact

The Compact attribute can be added to reduce the size and spacing of footnotes,
as well as remove their top border. This can be helpful when styling footnotes
in blog comments.

<Canvas>
  <Story name="Compact" args={{
    compact: true,
  }}>
    {(args) => template({
      items: [
        'Footnote 1',
        'Footnote 2',
        'Footnote 3',
        'Footnote 4',
      ],
      ...args
    })}
  </Story>
</Canvas>

## Template Properties

- `items`: An array of strings

## Template Blocks

- `content`: The footnote contents (overwrites anything placed in the items property)
